# Microservicios con NestJS

Este repositorio contiene múltiples proyectos desarrollados bajo una arquitectura de microservicios utilizando **NestJS**, **Prisma** y **Node.js**.

---

##  Tecnologías utilizadas

- NestJS
- Node.js
- Prisma ORM
- TypeScript
- REST & Microservices
- Arquitectura basada en Gateway

---

## 📁 Estructura del proyecto

### 📌 Examen
- CRUD completo de:
  - Lugares
  - Usuarios
  - Noticias
  - Eventos

### 📌 Proyecto Final
Arquitectura de microservicios:

- **client-gateway** → API Gateway
- **library-ms** → Microservicio de:
  - Libros
  - Préstamos
  - Miembros

### 📌 Tareas
Implementaciones independientes de microservicios:
- Control de usuarios
- Control de calidad
- Laboratorio
- Gestión de usuarios

### 📌 Prácticas
- Aplicaciones de productos con arquitectura distribuida
